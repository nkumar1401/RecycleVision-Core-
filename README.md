## üöÄ Key Features
* **High-Accuracy Sorting:** Powered by MobileNetV2 with a 90% validation accuracy.
* **12-Class Intelligence:** Recognizes Battery, Biological, Brown-glass, Cardboard, Clothes, Glass, Metal, Paper, Plastic, Shoes, Trash, and White-glass.
* **Instant Analysis:** Real-time classification and confidence scoring.
* **Actionable Logic:** Provides specific disposal instructions (Recyclable vs. Hazardous vs. Compostable).
